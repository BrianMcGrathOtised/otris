# Otris

Competitive multiplayer Tetris-style web game. 2-10 players compete in real-time — clear lines, send garbage, last one standing wins.

## Tech Stack

- TypeScript + HTML5 Canvas (frontend)
- Node.js + ws (backend)
- Vite (build)
- Vitest (testing)

## Getting Started

```bash
npm install
npm run dev
```
